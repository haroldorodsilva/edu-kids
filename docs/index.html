🔤 DigiLetras

Aprender a ler brincando

Aplicativo educacional gratuito para crianças de 3 a 8 anos aprenderem a ler e escrever em português brasileiro.

🚀 Abrir App ⭐ GitHub

✨ Funcionalidades

🎮

10 Jogos Educativos

Sílabas, quiz, memória, completar, escrever, primeira letra, montar frase, associação, colorir e histórias interativas.

🛤️

Trilhas por Idade

Trilhas gamificadas para 3-4, 5-6 e 7-8 anos com unidades temáticas, XP, estrelas e progresso persistente.

🎯

Modo Livre

Acesso direto a qualquer jogo com seleção de dificuldade e filtro de palavras por tema.

🤖

Histórias com IA

Gerador de histórias infantis usando IA. Suporta Groq (gratuito), Anthropic, OpenAI, DeepSeek e Gemini.

📱

PWA Offline

Funciona no navegador como app instalável. Sem necessidade de app store.

🔓

Open Source

Código aberto sob licença MIT. Contribua, adapte e distribua livremente.

🎮 Jogos

🔤 Sílabas ❓ Quiz ✏️ Completar 🧠 Memória ⌨️ Escrever 🅰️ Primeira Letra 📝 Montar Frase 🔗 Associação 🎨 Colorir 📖 Histórias

🏗️ Stack

React 19 TypeScript 5.9 Vite 8 Tailwind CSS 4 PWA Web Audio API Web Speech API fast-check vitest

🚀 Começar

git clone https://github.com/haroldorodsilva/digiletras.git
cd digiletras/digiletras
npm install
npm run dev

Acesse http://localhost:5173 no navegador.

Feito com ❤️ para as crianças brasileiras aprenderem a ler.
